# 设计系统变更日志

> Overnight Radar UI 设计系统版本更新记录

## [v1.0.0] - 2026-04-14

### 🎉 首次发布 - 初始稳定版

#### ✨ 新增

**色彩系统**
- 新增完整的 CSS 变量色彩系统（56个设计 token）
- 主色调：金色 #F59E0B（象征财富和信任）
- 强调色：紫色 #8B5CF6（体现科技和智能）
- 背景色：深蓝黑 #0F172A（深色专业主题）
- 金融色彩：红色上涨 / 绿色下跌（符合中国用户习惯）

**字体系统**
- 采用 IBM Plex Sans 作为主字体（专业金融字体）
- 8级字体大小：11px / 12px / 14px / 16px / 18px / 20px / 24px / 32px
- 4级字重：Normal(400) / Medium(500) / Semibold(600) / Bold(700)
- 3级行高：Tight(1.25) / Normal(1.5) / Relaxed(1.75)

**间距系统**
- 基于 4pt/8dp 标准的间距系统
- 10级间距：4px / 8px / 12px / 16px / 20px / 24px / 32px / 40px / 48px

**组件库**
- 按钮组件：Primary（主要）、Ghost（次要）
- 标签组件：金融数据涨跌标签
- 热力图区块：网格布局的数据展示
- 加载状态：旋转加载器、骨架屏
- 切换器：市场切换、指标切换
- 底部导航：Tab 导航组件
- 模态框：详情弹窗

**图标系统**
- SVG 矢量图标系统
- 统一的视觉风格（线性图标，2px 描边）
- **底部导航图标**（2026-04-14 更新）：
  - 市场观察：热力图方块（金色 #F59E0B）- 2x2 布局，表达热力图概念
  - 隔夜雷达：信号雷达（紫色 #8B5CF6）- 圆圈+中心点+8条扫描线，强调信号检测
- 常用图标：关闭、警告、上涨、下跌等

**动画系统**
- 标准化动画时长：150ms（快速）、200ms（标准）、300ms（慢速）
- 流畅的过渡效果
- 支持 reduced-motion 偏好设置

**响应式设计**
- 移动优先设计策略
- 5级断点：XS(<480px) / SM(480px+) / MD(768px+) / LG(1024px+) / XL(1440px+)

**可访问性**
- 符合 WCAG AA 标准（色彩对比度 ≥ 4.5:1）
- 完整的键盘导航支持
- 清晰的焦点状态（2px 金色轮廓）
- 语义化 HTML 结构
- ARIA 标签支持

#### 📄 文档

- UI-STANDARDS.md - 完整设计标准文档（14KB）
- UI-QUICK-REF.md - 开发快速参考指南（8KB）
- UI-REDESIGN-SUMMARY.md - UI 改造总结（8KB）
- design-system.html - 设计系统原型（20KB）
- VERSION.md - 版本管理文档
- CHANGELOG.md - 变更日志（本文件）

#### 🎨 设计原则

1. **数据优先**: 信息密度高，但保持可读性
2. **专业可信**: 金融级严谨，避免花哨设计
3. **快速响应**: 实时数据，流畅交互
4. **深色护眼**: 长时间使用不疲劳

#### 🔧 技术实现

- 原生 CSS（无预处理器）
- CSS 变量（支持主题定制）
- 原生 JavaScript（无框架依赖）
- Vite 5.0 构建工具

#### 🌍 本地化

- 中国用户习惯：红涨绿跌
- 中文字体优化
- 中文文案和标签

## 📋 版本规范说明

### 版本号格式

```
v主版本.次版本.修订版本
```

- **主版本 (Major)**: 重大架构变更，不兼容的修改
- **次版本 (Minor)**: 新功能，向后兼容
- **修订版本 (Patch)**: Bug 修复，小改进

### 变更类型标识

- ✨ **新增**: 新功能
- 🔄 **变更**: 现有功能的变更
- 🐛 **修复**: Bug 修复
- 🗑️ **删除**: 功能删除
- ⚡ **性能**: 性能优化
- 📝 **文档**: 文档更新
- 🎨 **设计**: UI/UX 改进
- ♿ **可访问性**: 可访问性改进
- 🔒 **安全**: 安全相关修复
